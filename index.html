Troubleshooting 🔧

😕 PC doesn’t recognize the headset?
– Make sure the Meta Quest PC App is installed and running.
– If using Link Cable: try a different USB 3.0 port.
– If using Air Link: ensure both PC and Quest are on the same 5GHz Wi-Fi network.

🖥️ Performance issues?
– Close other programs in the background.
– Lower the graphics settings in the Meta Quest PC App.
– Use a wired Link connection if Wi-Fi is unstable.

🔊 No sound?
– In Windows, check that the output device is set to Oculus Virtual Audio while in VR.

👉 Still stuck? Feel free to ask in the Discord channel for help.

Hilfe bei Problemen 🔧

😕 PC erkennt das Headset nicht?
– Stelle sicher, dass die Meta Quest PC-App installiert und gestartet ist.
– Bei Nutzung des Link-Kabels: versuche einen anderen USB-3.0-Port.
– Bei Air Link: achte darauf, dass PC und Quest im selben 5GHz-WLAN sind.

🖥️ Leistungsprobleme?
– Schließe andere Programme im Hintergrund.
– Reduziere die Grafikeinstellungen in der Meta Quest PC-App.
– Verwende eine Kabelverbindung, wenn das WLAN instabil ist.

🔊 Kein Ton?
– Überprüfe in Windows, ob das Ausgabegerät auf Oculus Virtual Audio eingestellt ist, während du in VR bist.
